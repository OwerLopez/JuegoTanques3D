# JuegoTanques3D — Unity Tank Battle

![Unity](https://img.shields.io/badge/Unity-3D-000000?style=for-the-badge&logo=unity)
![C#](https://img.shields.io/badge/C%23-Gameplay-178600?style=for-the-badge)

3D tank battle game built in **Unity** — movement, aiming and projectile combat implemented in C#.

## Highlights

- Tank controller: movement + turret aiming
- Projectile physics and impact handling
- Scene-based level design

## Run it

1. Open the project with **Unity Hub** (Unity 2021+).
2. Load the main scene from `Assets/Scenes`.
3. Press ▶ Play.
